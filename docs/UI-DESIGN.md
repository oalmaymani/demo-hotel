# UI Design — الموسمين للشقق المفروشة

## Brand / Feel
فندقي، هادئ، نظيف، فخم بدون تكلف.

## Colors (Tailwind-friendly)
- Primary (Olive): `#1F3B2C`
- Accent (Gold): `#C8A24A`
- Background: `#F7F6F2`
- Surface: `#FFFFFF`
- Text: `#111827` (gray-900), Secondary: `#374151` (gray-700)

## Typography
- Arabic: **IBM Plex Sans Arabic**
- English: **Inter**
- Headings: وزن 700، Body: 400/500

## Layout
- Max width: 1200px
- 12-column grid
- Cards: radius 16px, shadow خفيف
- Buttons: radius 14px, ارتفاع 44px للموبايل

## Components
- Navbar ثابت بسيط
- Hero مع CTA واضح
- Search bar (pill-style)
- Unit cards: صورة + عنوان + أيقونات + سعر
- Admin: جدول واضح + Chips للحالة

## RTL/LTR
- Arabic RTL تلقائي
- English LTR
- Toggle في الـ navbar (AR / EN)

## Imagery
- صور إضاءة طبيعية
- خطوط مستقيمة، ترتيب Minimal
